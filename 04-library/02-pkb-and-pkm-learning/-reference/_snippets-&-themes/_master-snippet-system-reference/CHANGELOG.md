## Migration Guide
### From Style Settings Plugin
If you were using the Style Settings plugin with a previous theme:
1. **Backup your settings:** Export your Style Settings configuration
2. **Install snippets:** Follow QUICKSTART.md
3. **Customize:** Adjust colors in `01-variables.css` to match your preferences
4. **Disable Style Settings:** These snippets replace Style Settings functionality
### From Individual Snippets
If you were using individual CSS snippets:
1. **Backup:** Copy your existing snippets folder
2. **Remove old snippets:** Delete conflicting files
3. **Install new package:** Copy all files from this collection
4. **Review changes:** Check QUICKSTART.md for new features
5. **Customize:** Merge any custom styles you had
