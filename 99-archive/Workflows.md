

### Permanent Note during Active Reading
1. **Actively reading**, comes across [[wiki-link]] that has *no file associated with it*.
2. **Determine and Locate** the folder that the *report your actively reading* is *located*.
3. Ensure that it is in *appropriate Folder* Location.
	  - **Note**: When the *new note* is created *Templater* will *generate* the appropriate information using the *stored template*, *based on which Folder* the new note is *being created in*.
4. If report your reading is *in correct folder*, proceed to *create* the new Note using the **hotkey**: `Ctrl+N`
5. As the note is being generated *Templater* will supply correct *information structure* based on the *pre-determined folder arrangement*.
6. The new **Permanent Note** will be created and will *prompt you* for these items:
	1. "**Enter concept/theory/person name**:"
	2. "**Enter first alias (or press Enter to skip)**:"
	3. "**Enter second alias (or press Enter to skip)**:"
7. After *filling out* the information *pop-ups*, the new permanent note will *automatically open* for you.
8. Perform a *quick* once over for *any errors in formatting*. **Fix if needed**.
9. *Copy Term* or Theory *from Title*, and manually *paste* it into the *Word Wise UI*. **Search for the definition**.
10. *Copy* the definition from the *Word Wise UI* and *paste* it into the newly created **Permanent Note**. In the *section* marked for *Definitions*.
11. *Add in* any other *relevant information* from the *report* 
12. *Add* the *source* to the note. **Which is the Active Reading Report you're using**.
