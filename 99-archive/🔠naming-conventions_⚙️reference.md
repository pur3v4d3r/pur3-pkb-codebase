# AI related Naming Conventions

## Prompts

**Naming Convention:**
`Prompt_{describe-the prompts purpose/what it accomplishes}_{the prompts ID number goes here.}`

**Example:**
`Prompt_Engineering-Technique-Academic-Review_IU3HJ6C2X6`

## Prompt-Templates

**Naming Convention:**
`Prompt-Template_{describe-the templates purpose/what it accomplishes}_{unique ID number goes here}`

**Example:**
`Prompt-Template_Prompt-Engineering-Technique-Academic-Review_IU3HJ6C2X6`

## Report Naming

**Naming Convention:**
`Report_{Entera description here}_{Enter the name of the model you used}`

**Example**
`Report_Self-Consistency-Chain-of-Thought-(CoT-SC)_Gemini-2.5_Deep-Research`

# Naming Conventions for My Different Notes

## Scratchpad Naming Convention

**Naming Convention:**
`Scratchpad-{Enter-Number-of-Scratchpad}`

**Example:**
`Scratchpad-02`

**Note:** *I have since automated this process now you use Templater to automatically create the Scratchpad and name it correctly using a Template.*
**The Hotkey is:** `Ctrl+Shift+(-)`
